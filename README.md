# Лабораторная работа №3: Снижение размерности данных

## Описание

В данной лабораторной работе исследуются методы снижения размерности для анализа многомерных данных. Основное внимание уделяется:

- Проблеме высокой размерности и её влиянию на анализ данных.
- Линейным (PCA) и нелинейным (t‑SNE, UMAP) методам снижения размерности.
- Выявлению скрытой структуры данных и интерпретации новых признаков.

В качестве набора данных используется **Bank Marketing Dataset** (`bank-additional-full.csv`), содержащий информацию о маркетинговых кампаниях банка.

## Цели работы

1. Изучить геометрию данных в пространстве признаков.
2. Реализовать алгоритм PCA с нуля (только NumPy) и сравнить с реализацией из `sklearn`.
3. Интерпретировать главные компоненты.
4. Визуализировать данные в исходном пространстве и после PCA.
5. Применить нелинейные методы t‑SNE и UMAP, исследовать влияние их параметров.
6. Исследовать степень сжатия данных через восстановление (MSE) и выбрать оптимальное число компонент.
7. Сформулировать выводы о применимости методов снижения размерности.
